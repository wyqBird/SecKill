# SecKill

## 技术点介绍
* 语言：Java
* 工具：IDEA
* 前端：Thymeleaf+Bootstrap+JQuery
* 后端：SpringBoot+JSR303+Mybatis
* 中间件：RabbitMO+Redis+Druid

## 内容
* 分布式Session
* 商品列表页
* 商品详情页
* 订单详情页
* 系统压测
* 缓存优化
* 消息队列
* 接口安全
